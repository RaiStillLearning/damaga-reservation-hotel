# Damaga Suite – Frontend System

Frontend application untuk sistem manajemen hotel **Damaga Suite**.  
Aplikasi ini dibangun menggunakan Next.js dan terhubung dengan backend Express melalui RESTful API.

## Tech Stack
- Next.js
- React.js
- JavaScript
- HTML5
- CSS3
- RESTful API

## Features
- Dashboard manajemen hotel
- Tampilan data kamar dan reservasi
- Integrasi API backend
- Routing dan struktur halaman menggunakan Next.js
- Desain responsif

School Project
