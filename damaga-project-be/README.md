# Damaga Suite – Backend System

Backend system untuk aplikasi manajemen hotel **Damaga Suite**.  
Project ini dikembangkan sebagai proyek sekolah dengan fokus pada pengelolaan data hotel dan penyediaan RESTful API untuk frontend.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JavaScript
- RESTful API

## Features
- Manajemen data kamar hotel
- Manajemen data reservasi
- Manajemen data pengguna (admin / staff)
- RESTful API untuk kebutuhan frontend
- Struktur backend terpisah dan terorganisir

## Project Type
School Project

## API Endpoints (Example)
- `GET /api/rooms` – Menampilkan daftar kamar
- `POST /api/rooms` – Menambahkan data kamar
- `PUT /api/rooms/:id` – Mengubah data kamar
- `DELETE /api/rooms/:id` – Menghapus data kamar
- `GET /api/reservations` – Menampilkan data reservasi
